About the application

The main use case of the project is the evaluation of databases performance, which stores invoice data in JSON/JSONB formats. Application’s business logic consist of recording and evaluating the performance of both PostgreSQL and MongoDB when processing invoices in JSON format. For outputting performance of both databases, a simple web interface is created.
